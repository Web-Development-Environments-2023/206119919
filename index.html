Questions


Question #1

Three main differences between web1, web2, and web3: Firstly, web1 is a static web which provides basic information and has limited user interactivity. On the other hand, web2 is a dynamic web that allows user interaction and collaboration, such as multimaedia content creation and social networking. Lastly, web3 is decentralized web that uses blockchain technology to create trustless systems and to enable decentralized applications, which named dApps.
Secondly, web1 and web2 relying on centralized servers and have a client-server architecture, where the server provides the content and the client consumes it. Conversely, web3 uses a peer-to-peer(P2P) network where every node can store and share data, enabling decentralized data storageand retrieval.
Lastly, web1 and web2 rely on traditional web technologies like HTML, JavaScript and CSS, whereas web3 uses blockchain technologies like smart contracts, decentralized file storage identity management.

Question #2

Use case for example: Supppose a user wish to search for "best coding bootcamps in a certain city" On web1, the user might go to a static website that lists the top coding bootcamps in a particular city based only on information from other bootcamps's websites, or other online related websites. The user can view and read the rankings or brief descriptions of each bootcamp, but cannot contribute their own reviews or ratings.
On web2, the user moght access a dynamic website or platform, such as a SwitchUp or Course Report, that allows them to view coding bootcamp rankings and read reviews from other users who have attended the bootcamps. Additionaly, the user contribute their own reviews, ratings, and recommendations for bootcamps, as well as interact with other users by leaving comments on reviews and sharing their own experiences.
On the other hand, on web3, the user might access a dApp (i.e. decentralized application) that allows users to contribute and access information about coding bootcamps in the particular city. The dApp might use a blockchain-based system to motivate users to share rheir own reviews, recommendations and ratings for bootcamps. Users can interact with each other, leave comments and share their own experiences in more detail, including projects they worked on and job offers they received after graduating the bootcamp. The information on the dApp is decentralized and more trustworthy since it's based on user-generated content and motivated by blockchain rewards.
Therefore, the user experiance and features available on the user when searching for a different query, such as coding bootacamps on a certain city, can also vary significantly depending on which type of web tech the user is on.
